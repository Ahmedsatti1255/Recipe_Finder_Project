# Recipe_Finder_Project
Recipe Finder is a web app that uses the Google Search API to find recipes across the web. Enter a dish or ingredients to get real-time links from various cooking sites. It offers fast, diverse, and up-to-date results without relying on a fixed database, making recipe discovery simple and efficient.
