# 🍽️ Recipe Finder

Recipe Finder is a web app that uses the **Google Search API** to find recipes across the web. Enter a dish or ingredients to get real-time links from various cooking sites. It offers fast, diverse, and up-to-date results without relying on a fixed database, making recipe discovery simple and efficient.

---

## ✨ Features

- 🔍 Search by dish name or ingredients
- ⚡ Real-time results via Google Search API
- 🌐 Links from diverse cooking sites and food blogs
- ❤️ Save your favorite recipes
- 🔐 User login support
- 📦 No fixed database — always fresh content

## 🚀 Getting Started

1. Clone the repository
```bash
   git clone https://github.com/your-username/Recipe_Finder_Project.git
```
2. Add your Google Search API key to the config
3. Open `index.html` in your browser
